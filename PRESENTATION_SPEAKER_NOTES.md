### 为什么这个项目重要？
```

### 为什么是多平台？
```
现象: 用户在不同工具工作 (VS Code, ChatGPT, Claude, Web)
问题: 每个平台都需要单独配置
解决: 一份数据 (digitaltwin.json) → 4 个平台同步
```

### 为什么是 AI 代理？
```
AI 不是简单的聊天机器人
而是了解你的专业代理，可以:
- 提醒你遗忘的项目
- 指出你的技能缺口
- 预测面试官会问什么
- 帮你完美回答问题
```

```

### Q2: 为什么需要 4 个平台？
```
A: "不同的用户工作流不同：
   - 程序员用 VS Code
   - 其他人用 ChatGPT
   - 高级用户用 Claude
   - 快速用户用 Web

   关键是数据同步，无论在哪个平台都是同一个 Digital Twin。"
```

### Q3: 数据怎么保证安全？
```
A: "三层保护：
   1. 本地 .env 管理敏感信息
   2. Vercel 的标准安全措施
   3. 用户完全控制档案数据

   我们不会存储你的对话历史。"
```

### Q4: 能否商业化？

   - API 集成（第三方）
   
   但初心是帮助人们，不以盈利为主。"

### Q5: 未来计划是什么？
